# **App Name**: HavenFind

## Core Features:

- Find Shelter: Display a list of available shelters nearby, showing location, capacity and responsible organization.
- Send SOS Message: Send an SOS message with the user's location to a predefined emergency response system.
- Request Assistance: Allow users to call or chat with support teams to request assistance, such as medical aid or transportation. Generate canned responses for common requests using a tool connected to an LLM, to improve response time.
- Real-Time Route Navigation: Show the user the optimal route to a shelter on a map.
- Track Shelter Movement: Track the location of specific shelters (only available for admins)
- Weather Forecast Alerts: Displays real-time weather alerts for potential floods or natural disasters
- Monitor Driver Movements: Allow the implementing organization to monitor real-time movements of drivers to better coordinate evacuation efforts.

## Style Guidelines:

- Primary color: Soft, calming blue (#64B5F6) to inspire trust and safety, drawing from the concept of providing refuge.
- Background color: Light, desaturated blue (#E3F2FD), providing a calm, unobtrusive backdrop.
- Accent color: Muted green (#81C784), an analogous color that represents hope and renewal, contrasting enough for important calls to action.
- Body and headline font: 'PT Sans', a clear and modern sans-serif for ease of reading and accessibility.
- Use simple, clear icons to represent key functions. Consider icons that are universally understandable to aid users with diverse language skills.
- Maintain a clean and intuitive layout. Prioritize essential information and features prominently to reduce cognitive load in stressful situations.
- Use subtle animations for transitions and feedback to enhance user experience without distracting from critical information.
